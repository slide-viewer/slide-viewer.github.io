<slide-viewer> Web Component

GitHub source

Displays a series of slides with optional prev/next buttons, slide indicator and a scrollbar. The slides can be images or any HTML content. NOT using CSS animations, scrollspeed is the standard browser scrollspeed

play=1000 | Auto play every 1000ms | 3000 nobutton | Hides the prev/next buttons scrollbar | Shows the scrollbar | noscrollbar dots=0 | sets dot size in pixels | 15 rtl | Right to left start at the last slide noreverse | No reverse loop hold | cursor holds current image | automatic
